# Rock Paper Scissors Simulator (Statistics Mode)

## Description
This project is a console-based Rock Paper Scissors simulator written in C++.  
It allows a player to play multiple rounds against the computer while tracking detailed statistics such as wins, losses, draws, and probability percentages.

The project focuses on simulation logic, randomness, and basic statistical analysis.

## Features
- Player vs computer gameplay
- Random computer choice generation
- Win, loss, and draw tracking
- Total rounds played
- Win, loss, and draw percentage calculations
- Computer choice distribution statistics
- Input validation
- Console-based interface

## Skills Demonstrated
- C++ fundamentals
- Loops and conditional logic
- Random number generation
- Statistical tracking and probability calculations
- Console input/output handling
- Program state management

## How to Run
1. Open the project in a C++ IDE (Code::Blocks, Visual Studio, or similar).
2. Compile the program.
3. Run the executable.
4. Enter your choice:
   - `0` for Rock
   - `1` for Paper
   - `2` for Scissors
5. Play multiple rounds to observe how the statistics change.
